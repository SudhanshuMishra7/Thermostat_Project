\# Thermostat Control System



This project is a simple thermostat simulation using MATLAB/Simulink.  

It controls temperature using ON/OFF logic based on a setpoint.



\## Files

\- thermostat\_model.slx → Simulink model  

\- clear.m → Clears workspace  

\- workspace.mat → Saved workspace data  



\## Working

\- If temperature is below setpoint → Heater ON  

\- If temperature reaches setpoint → Heater OFF  



\## Software Used

\- MATLAB

\- Simulink
